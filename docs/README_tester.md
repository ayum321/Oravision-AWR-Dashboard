# OraVision AWR Pro — Tester Setup

## Requirements
- Windows 10/11
- Python 3.10 or later → https://python.org/downloads (tick "Add Python to PATH" during install)

## How to Run

1. Unzip the folder anywhere (e.g. `C:\Tools\awr-dashboard\`)
2. Double-click **`start.bat`**
3. Browser opens automatically at `http://localhost:8000`
4. Upload your AWR HTML file and analyse

## First run
The first run installs dependencies (~30 seconds). Subsequent runs start instantly.

## Stopping
Press `Ctrl+C` in the terminal window, then close it.

## Supported AWR formats
- Oracle 11g, 12c, 19c, 21c
- AWR HTML report (generated by `awrrpt.sql`)
- Single report or two reports for comparison
